# アーキテクチャ概要

Agent Hubは、拡張性と柔軟性を考慮して設計されたモジュラーアーキテクチャに従っています。その中核では、エージェント管理のさまざまな側面を処理するためにマイクロサービスアプローチを活用しています。

## システムコンポーネント

### コンテンツレイヤー
すべてのチュートリアルとドキュメントはMDXファイルとして保存され、TinaCMSを通じて簡単な編集とバージョン管理が行われます。

### フロントエンド
Next.js App Routerで構築されており、SEOのためのサーバーサイドレンダリングと、スムーズなユーザー体験のためのクライアントサイドのインタラクティブ性を備えています。

### 検索
Oramaによって駆動される全文検索エンジンで、ブラウザ内で完全に動作し、即座に結果を提供します。

## データフロー

1.  コンテンツはCMS管理インターフェースを通じて作成されます
2.  MDXファイルはGitリポジトリに保存されます
3.  Next.jsがビルド時またはリクエスト時にページをレンダリングします
4.  クライアントサイドの検索がコンテンツをインデックス化し、即時検索を可能にします
