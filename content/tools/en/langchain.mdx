# LangChain

LangChain is a framework for developing applications powered by language models. It enables applications that are context-aware and can reason about their environment.

## Key Features

- **Composable Components**: Chain together LLM calls, tools, and data sources
- **Agent Support**: Build autonomous agents that can use tools and make decisions
- **Memory Management**: Persist conversation state across calls
- **RAG Capabilities**: Integrate retrieval-augmented generation with any vector store
- **Streaming**: First-class support for streaming responses

## Use Cases

- Chatbots with document retrieval
- Autonomous research agents
- Data analysis pipelines
- Code generation and review
