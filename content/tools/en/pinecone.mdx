# Pinecone

Pinecone is a fully managed vector database that makes it easy to add high-performance vector search to your AI applications.

## Key Features

- **Fully Managed**: No infrastructure to manage
- **High Performance**: Sub-millisecond query latency at any scale
- **Hybrid Search**: Combine vector and keyword search
- **Metadata Filtering**: Filter results by metadata attributes
- **Real-time Updates**: Index and query in real-time

## Use Cases

- Semantic search engines
- Recommendation systems
- RAG pipelines
- Image and audio similarity search
