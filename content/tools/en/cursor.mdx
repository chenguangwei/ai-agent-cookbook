# Cursor

Cursor is an AI-first code editor designed to make you extraordinarily productive. It features deep AI integration for code generation, editing, and understanding.

## Key Features

- **AI Chat**: Ask questions about your codebase in natural language
- **Code Generation**: Generate code from prompts with full context awareness
- **Smart Editing**: AI-powered code transformations and refactoring
- **Codebase Understanding**: AI understands your entire project structure
- **Multi-model Support**: Use GPT-4, Claude, and other models

## Use Cases

- Rapid prototyping
- Code refactoring
- Bug fixing with AI assistance
- Learning new codebases
