# LangChain

LangChain 是一个用于开发由语言模型驱动的应用程序的框架。它支持构建能够感知上下文并对其环境进行推理的应用程序。

## 核心特性

- **可组合组件**：将 LLM 调用、工具和数据源链接在一起
- **智能体支持**：构建能够使用工具并自主决策的智能体
- **记忆管理**：在多次调用间持久化对话状态
- **RAG 能力**：与任何向量存储集成，实现检索增强生成
- **流式响应**：提供一流的流式响应支持

## 应用场景

- 支持文档检索的聊天机器人
- 自主研究智能体
- 数据分析流水线
- 代码生成与审查
