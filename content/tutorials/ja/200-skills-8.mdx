# 半月で200以上のスキルを試した結果、これら8つが最も重要。最後の1つは絶対に必要です

インストール、アンインストール、再インストール。2週間いじくり回した結果、本当に必要なのはこの8つだけだとわかりました。

機能が一番多いわけではありませんが、私の最も現実的な悩みを解決してくれます：QRコードログインが不要、繰り返しのWeb操作が不要、エージェントが忘れる心配が不要、7〜8体のエージェントの状態を手動で管理する必要がありません。

各スキルの後にインストールコマンドを書いておきました。コピーして使ってください。

## agent-browser — エージェントにWebを操作させる

ページをスクレイピングしたい？「[ウェブサイト]にアクセスして、[リスト]を取得して」と言うだけ。エージェントが自分でブラウザを開き、ページを操作し、構造化されたデータを返します。環境設定、スクレイピング対策、コードのメンテナンスは不要です。

```text
npx skills add vercel-labs/agent-browser@agent-browser
```

## bb-browser — ログイン不要の魔法のツール

以前、エージェントに小紅書に投稿させたい場合、まずQRコードをスキャンしてログインし、セッションが切れたら再スキャンする必要がありました。bb-browserは直接ローカルブラウザの状態を使用します—あなたはすでにログイン済みで、エージェントはあなたのIDを使って操作します。小紅書に投稿するたびにQRコードをスキャンする必要はありません。

```text
npx skills add epiral/bb-browser@bb-browser
```

## OpenCLI — 18プラットフォームをカバーするワンコマンド操作

コメント、フォロー、検索などの手動操作をコマンドラインインターフェースに変換します。Bilibili、知乎、小紅書、X、Reddit、GitHubなど18のプラットフォームをカバー。新しいWebページをCLIに変換する自己反復も可能です。

```text
npx skills add joeseesun/opencli-skill@opencli

```

## skill-vetter — 新しいスキルをインストールする前のセキュリティチェック

スキルマーケットは豊かになる一方で、リスクも増えています。skill-vetterはインストール前にスキャン：ネットワーク外向き呼び出し、機密環境変数の読み取り、システムディレクトリへの書き込み、不審なbase64デコードをチェックします。

```text
npx skills add useai-pro/openclaw-skills-security@skill-vetter

```

## self-improving-agent — エージェントに反省から学ばせる

あなたが修正したミス、遭遇した落とし穴、頻繁に言及する作業習慣を自動的に記録します。その後、それらを「知識カード」に整理し、後続のタスクで積極的に使用します。反省しないエージェントは、どれだけ長く訓練しても初心者のままです。

```text
npx skills add charon-fan/agent-playbook@self-improving-agent

```

## lossless-claw — エージェントに長期記憶を与える

OpenClaw作者推奨のソリューション。すべての会話をデータベースに永続的に保存し、バックグラウンドで古い会話をツリー構造の要約にパッケージ化してトークン消費を制御します。あなたのエージェントはもう「忘れ」ません。

```text
npx skills add Martian-Engineering/lossless-claw
```

## control-center — あなたのエージェントを管理するダッシュボード

7〜8体のエージェントを管理していて、それぞれが沈黙したら—生きているのか死んでいるのかわかりません。このパネルが解決します：各タスクが消費したトークン数を確認、各エージェントが正常か詰まっているかチェック、各エージェントのメモリ、ペルソナ、タスクドキュメントを表示/変更できます。

```text
npx skills add aradotso/trending-skills@openclaw-control-center

```

## openclaw-backup — あなたのエージェントの最終的な安全網

エージェント全体の設定ファイルとメモリデータベースをスケジュールバックアップ。誤ってエージェントを「殺して」しまっても、復元できます。設定、メモリ、過去の会話—すべてワンクリックでロールバック。

```text
npx skills add theagentservice/skills@openclaw-backup
```

ツールの価値は、どれだけ多くの機能があるかではなく、あなたのどの悩みを解決するかにあります。

この8つのスキルは、2週間いじくり回し、200以上を試した結果、私が本当に残したものです。

あなたの役に立つことを願っています。
