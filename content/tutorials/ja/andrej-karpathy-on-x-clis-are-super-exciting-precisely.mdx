# Andrej Karpathy (@karpathy)

_2026-02-24T14:36:34.000Z_

CLIはまさに「レガシー」技術であるがゆえに非常にエキサイティングです。なぜなら、AIエージェントがネイティブかつ容易にそれらを使用し、組み合わせ、ターミナルツールキット全体を通じて対話できるからです。

例えば、あなたのClaude/Codexエージェントにこの新しいPolymarket CLIをインストールし、任意のダッシュボードやインターフェース、ロジックを要求してみてください。

エージェントがあなたのために構築してくれます。

Github CLIもインストールすれば、リポジトリのナビゲート、issue、PR、ディスカッション、さらにはコードそのものを確認するよう依頼できます。

例：Claudeは約3分で、取引量の多いPolymarketと24時間の変動を示すこのターミナルダッシュボードを構築しました。

あるいは、それをWebアプリにすることも、あなたが望むどんなものにでもできます。

より大きなパイプラインのモジュールとして使用するときには、さらに強力になります。

もし何らかの製品やサービスをお持ちなら考えてみてください：エージェントはそれらにアクセスして使用できますか？ - あなたの（人間向けの）レガシードキュメントは、少なくともMarkdownでエクスポート可能ですか？ - あなたの製品向けにスキルを書いていますか？ - あなたの製品/サービスはCLI経由で使用可能ですか？

あるいはMCP経由では？ - ...

今は2026年です。

エージェントのために。

構築しましょう。
