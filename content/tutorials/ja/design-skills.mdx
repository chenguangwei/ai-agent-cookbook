## 現在市場で入手可能なデザインスキルの別のまとめです：

https://x.com/dingyi/status/2033679210766843928

1.  Impeccable
    17のデザインコマンド（/polish, /audit, /distill など）を備えた強化版フロントエンドデザインスキル。
    https://t.co/Vo6UIAcDvu

2.  UI Skills
    15の独立したスキルを含む：baseline-ui（Tailwindデザインの一貫性）、fixing-accessibility（アクセシビリティ監査）、fixing-metadata（SEOメタデータ）、fixing-motion-performance（アニメーションパフォーマンス）、12-principles-of-animation（ディズニーアニメーション原則）など。
    https://t.co/3X5o3UyX3l

3.  Taste Skill
    taste-skill（フロントエンドデザイン）、redesign-skill（既存プロジェクトのアップグレード）、output-skill（AIに完全なコードを書かせる）、soft-skill（高級感のある見た目と質感を実現。プレミアムフォント、ゆとりのある余白、レイヤードカードデザイン、滑らかなスプリングアニメーション、フローティングナビゲーションをカバー。）を含む。
    https://t.co/bElD1T4EVZ

4.  Better Icons
    200,000以上のアイコンを検索・取得するためのMCPサーバー＋スキル。一般的なアイコンライブラリの自動学習と、プロジェクトのアイコンファイルへの直接同期をサポート。
    https://t.co/dF2iFCHazY

5.  UI Design Brain
    レイアウトパターン、アクセシビリティルール、アンチパターンを含む、60以上のコンポーネントベストプラクティスのライブラリ。5つのデザインスタイル（SaaS、ミニマリスト、エンタープライズ、クリエイティブ、データダッシュボード）をサポート。
    https://t.co/yHVPZUgzXB

6.  AI kit by Motion ($299)
    Motion+メンバー専用。アニメーションエキスパートスキル、パフォーマンス監査、CSSスプリング生成、トランジション可視化、ドキュメント検索ツールを含む。
    https://t.co/H48KM7V4j8

## 7. UI UX Pro Max

https://x.com/dingyi/status/2033679223140061222

50以上のデザインスタイル、97のカラースキーム、9つの技術スタックを含み、包括的なUI/UXデザインインテリジェンスを提供。
https://t.co/tlXF5HeC6f

8.  Anthropic frontend-design
    Anthropic公式スキルリポジトリ。クリエイティブデザイン、技術開発、エンタープライズコミュニケーションなどのカテゴリにわたるサンプルスキル、およびドキュメント処理スキル（docx, pdf, pptx, xlsx）を含む。
    https://t.co/yRZbPGmg8A

9.  Vercel web design guidelines
    Vercel公式スキルコレクション。web-design-guidelinesは100以上のUI監査ルールを含む。
    https://t.co/7j3T3D88eW

10. Designer Skills Collection
    63のスキルと27のコマンドをカバーし、8つのプラグインを含む：ユーザーリサーチ、デザインシステム、UX戦略、UIデザイン、インタラクションデザイン、プロトタイプテスト、デザインオペレーション、デザインツールキット。
    https://t.co/nLzppuBB5N

11. Emil Skills ($99)
    https://t.co/7hkOrS5oJ2 がリリースしたMotionスキル。別途購入が必要。
    https://t.co/ApaiNlibvj

12. Skills by InterfaceCraft ($199)
    このコースを購入する必要があります。
    https://t.co/HmM0n8xc1k
