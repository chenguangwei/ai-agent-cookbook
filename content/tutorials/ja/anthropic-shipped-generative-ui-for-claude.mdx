## Anthropic、Claude向け生成UIをリリース。その仕組みをリバースエンジニアリングしてみると…

https://x.com/micLivs/status/2032244251464188184

AnthropicがClaude向け生成UIをリリースしました。その仕組みをリバースエンジニアリングし、PI向けに再構築してみました。

会話エクスポートから完全なデザインシステムを抽出。morphdomによるDOM差分処理を用いて、HTMLをネイティブmacOSウィンドウにライブストリーミングしています。

記事: https://t.co/BcYo94YqK3
リポジトリ: https://t.co/EfMDX58NWc

@badlogicgames の pi と @DanielGri の Glimpse を基盤として構築。
