## Another summary of the design skills currently available on the market:

https://x.com/dingyi/status/2033679210766843928

1.  Impeccable
    An enhanced frontend-design skill with 17 design commands (/polish, /audit, /distill, etc.).
    https://t.co/Vo6UIAcDvu

2.  UI Skills
    Contains 15 independent skills: baseline-ui (Tailwind design consistency), fixing-accessibility (accessibility audit), fixing-metadata (SEO metadata), fixing-motion-performance (animation performance), 12-principles-of-animation (Disney animation principles), etc.
    https://t.co/3X5o3UyX3l

3.  Taste Skill
    Contains taste-skill (frontend design), redesign-skill (upgrading existing projects), output-skill (forcing AI to write complete code), soft-skill (making things look and feel expensive. Covers premium fonts, generous whitespace, layered card designs, smooth spring animations, and floating navigation.).
    https://t.co/bElD1T4EVZ

4.  Better Icons
    MCP server + skill for searching and fetching 200,000+ icons. Supports automatic learning of common icon libraries and direct syncing to project icon files.
    https://t.co/dF2iFCHazY

5.  UI Design Brain
    A library of 60+ component best practices, including layout patterns, accessibility rules, and anti-patterns. Supports 5 design styles (SaaS, Minimalist, Enterprise, Creative, Data Dashboard).
    https://t.co/yHVPZUgzXB

6.  AI kit by Motion ($299)
    Exclusive for Motion+ members. Includes an animation expert skill, performance audit, CSS spring generation, transition visualization, and documentation search tools.
    https://t.co/H48KM7V4j8

    https://t.co/nLzppuBB5N

11. Emil Skills ($99)
