# Andrej Karpathy (@karpathy)

_2026-02-24T14:36:34.000Z_

CLI 之所以超级令人兴奋，恰恰因为它们是“遗留”技术，这意味着 AI 智能体可以原生且轻松地使用它们、组合它们，并通过整个终端工具包与之交互。

例如，让你的 Claude/Codex 智能体安装这个新的 Polymarket CLI，并要求它生成任意的仪表盘、界面或逻辑。

智能体会为你构建出来。

再安装 Github CLI，你就可以让智能体浏览代码库、查看问题、PR、讨论，甚至代码本身。

示例：Claude 在约 3 分钟内构建了这个终端仪表盘，展示了交易量最高的 Polymarket 市场及其 24 小时变化。

或者你也可以把它做成 Web 应用，或任何你想要的形式。

当将其用作更大流程的模块时，其能力会更加强大。

如果你有任何产品或服务，请思考：智能体能否访问并使用它们？  
- 你那些面向人类的遗留文档是否至少能以 Markdown 格式导出？  
- 你是否为你的产品编写了技能（Skills）？  
- 你的产品/服务能否通过 CLI 使用？

或者通过 MCP？  
- ……

现在是 2026 年。

智能体。

而构建。
