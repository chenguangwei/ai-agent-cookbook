# 卧丢，Obsidian 一句话画图神器，1600星


这个 Obsidian 神器我自己一直都在用！！！

你们有没有这种痛苦：看完一篇文章想画个思维导图，打开各种工具折腾半天，画出来还丑得一批？？？

今天分享一个 GitHub 上 1600 星的开源项目，叫 axton-obsidian-visual-skills，一句话总结：在 Obsidian 里用自然语言说一句话，AI 直接帮你画出手绘风格的图！！！

流程图、思维导图、时间线、对比图……你说啥它画啥，而且画出来是那种手绘风格，不是死板的方框线条，真的太香了😂

![](https://pbs.twimg.com/media/HCZrgAvaIAA7df1.jpg)

## 三个技能，覆盖所有画图场景


这个项目其实是个技能包，里面塞了三个 Claude Code Skill：

1.

Excalidraw 图表生成器

这个是我最爱的！！！生成的图全是手绘风格，8 种图表类型随便选：流程图、思维导图、层级图、关系图、对比图、时间线、矩阵图、自由布局。

还支持中文！还能做动画！！！

2.

Mermaid 可视化器

6 种图表类型，自带语法纠错，不用担心写错 Mermaid 语法。布局可以选垂直还是水平，简洁还是详细，配色也是专业级的。

3.

Obsidian Canvas 创建器

直接生成 Obsidian 原生的 .canvas 文件，两种布局模式：思维导图式的放射状，或者自由定位。节点自动配色、自动连线、自动排版。

三个技能组合起来，基本上你能想到的图都能画！！！

![](https://pbs.twimg.com/media/HCZrgB1aMAEAASR.jpg)

## 安装简单到离谱


两种方式：

方式一：插件市场（推荐）

> /plugin marketplace add axtonliu/axton-obsidian-visual-skills
> /plugin install obsidian-visual-skills

两行命令，搞定。

方式二：手动安装

把项目 clone 下来，skill 文件夹丢到 ~/.claude/skills/ 里就行。

前提是你得有 Claude Code CLI 和 Obsidian，Excalidraw 插件也要装一下。

## 用起来有多爽？


直接跟 Claude 说人话就行：

- "帮我画个 CI/CD 流水线的流程图"
- "把这篇文章整理成思维导图"
- "用 Mermaid 画个用户注册的时序图"
- "做个 Canvas 对比 React 和 Vue 的优缺点"

它自己就给你生成好了，直接在 Obsidian 里打开就能看。手绘风格的图拿去做 PPT、写文档、发推文都特别好看🔥
