## 第一招：先给龙虾做「身份录入」，1 分钟搞定


这是新手最容易忽略的一步，也是决定你的龙虾是「人工智障」还是「专属助理」的最关键一步。

很多人装完 OpenClaw，上来就问 "你能干嘛""帮我写个东西"，但你连你是谁、你要干嘛、有什么喜好都没告诉它，它怎么可能给你想要的答案？OpenClaw 最牛的核心能力，就是永久记忆 。

说白了，这一步就是给你的龙虾做「岗前培训」，让它知道自己跟着谁干活，老板的喜好、习惯、工作节奏是啥，以后干活才能精准踩在你的点子上。

小白直接复制照着改，1 分钟搞定。 把下面这段信息发给龙虾就行，结尾一定要加一句：以上内容请永久记住，这是我的基本个人信息。

> 我叫 XX，今年 XX 岁，在 XX 城市做 XX 工作，主要负责 XX 内容。我平时不喜欢太正式的说话方式，你跟我沟通可以随意一点。我每天 XX 点上班，平时 XX 点睡觉。我目前在 XX / 有 XX 习惯，相关内容麻烦多提醒我

就这么几句话，不用写小作文，不用很正式。你还可以随时补充信息，今天告诉它你喜欢健身，明天告诉它你在学法语，这些内容它都会一一记下来，越用越懂你。 

## 第二招：装备你的龙虾，小白必装技能清单


先给小白一句话讲透，90% 的人都搞混的两个词：skill 和插件。

skill，就是龙虾能学会的本事 —— 比如帮你搜网页、发邮件、整理文档、控制浏览器，每一个 skill，就是一项它能直接给你干的活。

插件，就是给龙虾加个外设 —— 比如加个语音输入框、换个皮肤，属于锦上添花的东西，新手完全不用先碰。先把龙虾的本事练起来，再搞那些花里胡哨的。

## 第三招：进阶黄金组合，4 个玩法让龙虾 24 小时给你打工


很多人装了一堆技能，还是觉得龙虾不好用，为啥？因为你让它一个人干所有活，肯定干不好。进阶的核心，就是让 OpenClaw 当你的「总经理」，听懂你的想法，拆解任务，然后派给专业的工具去干，能力直接翻倍。

给你们整理了 4 个社区最火、小白也能直接上手的组合，照着做就行。

![](https://pbs.twimg.com/media/HCcxULSbwAA4l-z.jpg)

OpenClaw + Claude Code = 最强代码打工搭档

现在全网最火的组合，没有之一。给你们说清楚分工：Claude Code 是专门写代码的，改 bug、做项目，能力拉满，但它记不住你的喜好、不知道你的项目背景，每次用都要重新说一遍；而 OpenClaw 刚好能补上这个坑，它记得你所有的需求、项目历史、做事习惯。

OpenClaw + n8n = 自动化神器，重复活全让它干

n8n 是个开源的自动化工具，能打通你所有的软件，但是单独用要自己拖拽配置，门槛很高；和 OpenClaw 组合之后，你只用说人话，它就能帮你自动跑流程。

OpenClaw + Obsidian / 知识库 = 你的专属第二大脑

知识工作者、创作者必看。Obsidian 是很多人用来记笔记、搭知识库的工具，和 OpenClaw 打通之后，你跟龙虾聊天的所有灵感、想法、干货，它都会自动帮你整理进知识库，还能跟你之前的笔记做关联，不用你自己手动复制粘贴。

OpenClaw + Telegram / 飞书 = 龙虾随时随地跟着你

这个组合直接改变你用龙虾的方式。之前你只能坐在电脑前用龙虾，打通 Telegram 或者飞书之后，你用手机随时随地都能跟它说话，发文字、发语音、发图片，它都能处理，结果直接回到你聊天框里。

## 第四招：4 个真实可落地的🦞变现路径


很多人问我，把龙虾玩明白了，能不能靠它赚钱？当然能。现在已经有部分人靠这个变现了，从新手就能做的，到有技术含量的，全给你们整理好了。

技术服务 —— 很多老板、商家、自媒体，都想有自己的 AI 助理，但不会装、不会配、不会搭流程。你可以把自己的经验做成服务，帮客户搭建专属的龙虾工作流、定制行业方案

Agent 代运营 —— 现在很多品牌、商家，都想做自己的 AI Agent，但不会落地、不会运营。你可以用龙虾为核心，帮他们搭建、运营专属的 AI 助理，跑通一个客户，后续复购和转介绍根本不愁。
