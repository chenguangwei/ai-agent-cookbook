## 第一步，在 Cloudflare 中添加你的域名（免费）


1. 在[腾讯云](https://buy.cloud.tencent.com/domain?domain=ResearchWang13&tlds=.com&from=domain-console&position=overview-input-search&_t=1772266557608)中购买域名，选择最便宜的，我选的是space邮箱，只花费10r
2. 进入 [https://dash.cloudflare.com/](https://dash.cloudflare.com/)，使用google账号登入选择免费的方案
3. 点击页面中的 "Onboard a domain" 按钮，在输入框中输入 购买的域名以 （ResearchWang13.com）为例，注意：不带 www 或 http://。
4. 点击 continue 按钮，选择 free plan 
5. Cloudflare 会自动扫描你的域名是否有绑定过其他 IP，继续点击continue
6. 直到出现 xxxxx.ns.cloudflare.com ,yyyyy.ns.cloudflare.com ，该页面不要关闭
7. 这时候我们去[腾讯云管理页面](https://console.cloud.tencent.com/domain/all-domain/manage/domain-mb8bn4sq/basicinfo)中将域名解析修改地址，点击修改 DNS服务器 
8. 选择 “使用非腾讯云DNS” ，将CF提供的域名解析地址输入，等待提交成功
9. 返回cloudflare页面，下滑，点击 “I updated my nameservers” 按钮
10. 后面一路 跳过 ，知道你的[Cloudflare页面](https://dash.cloudflare.com/0e3f60448b5858ee72c3d3e03cbac4ac/home/domains) 显示该域名 active 即可

2. 出现很多操作系统，选择 docker ，复制下面命令行

3. 下滑，点击 next 
4. 域名选择自己购买的域名，Type 选择 Http ， 端口填写 app:8080，点击Complete setup

## 第三步，在服务器上将全部服务运行起来 (依次执行)


1. mkdir -p ~/ResearchWang13 && cd ~/ResearchWang13
2. vim docker-compose.yml
3. 将一下内容粘贴进docker-compose.yml文件中

```text
version: '3.8'

services:
  app:
    image: weishaw/sub2api:latest
    container_name: sub2api_app
    restart: always
    environment:
      # --- Neon 数据库 填你自己的---

      - DB_HOST=ep-gentle-boat-a1b92kbq.ap-southeast-1.aws.neon.tech
      - DB_PORT=5432
      - DB_USER=neondb_owner
      - DB_PASSWORD=
      - DB_NAME=neondb
      - DB_SSL=true

      # --- redis 缓存  填你自己的---

      - REDIS_HOST=chief-crayfish-12273.upstash.io
      - REDIS_PORT=6379
      - REDIS_PASSWORD=
      - REDIS_TLS=true

      # --- 基础安全配置 ---

      - JWT_SECRET=sk_sub2api_random_secret_key_please_change_me_to_be_safe
      - ADMIN_PASSWORD=
      - APP_PORT=8080
      - APP_ENV=production
      # --- WARP 代理 ---

      - HTTP_PROXY=socks5://172.17.0.1:40000
      - HTTPS_PROXY=socks5://172.17.0.1:40000
      - ALL_PROXY=socks5://172.17.0.1:40000

  # --- Cloudflare Tunnel 隧道 填你自己的---

  cloudflared:
    image: cloudflare/cloudflared:latest
    container_name: sub2api_tunnel
    restart: always
    command: tunnel --no-autoupdate run --token  你的zeroTurn的token
```

4.    最后运行 docker compose up -d ， 全部显示 Started 就成功了

## 最后一步！在浏览器中配置帐号池子


1. 进入 你买的域名 ，我这是 [ResearchWang13](https://researchwang13.space/home) .
2. 如果进入到 setup 界面 ，就按照上面代码的格式，将值都输入进去，记得要启动 SSL 权限
3. 点击左侧的分组管理，需要新建一个账号池的分组
4. 点击左侧的账号管理，点击添加账号，名字随意，选择 openai ，选择Oauth方式
5. 生成授权连接 ，在服务器的浏览器中打开 生成的链接，登入你已经加入business团队的账号，会生成token，将token填入，点击完成授权
6. 测试：在账号管理页面，每个账号右侧都有 ... 点击测试链接，选择模型 我选的是 GPT - 5.3 Codex , 测试成功即可
7. 生成专属的apikey，点击左侧 “API密钥” ，创建密钥，注意额度填0
