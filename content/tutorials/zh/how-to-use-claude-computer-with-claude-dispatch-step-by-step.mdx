# 1. 如何访问‘Claude Computer’

唯一的访问方式：

1. 仅限Mac（Windows版即将推出，敬请期待）。
2. 下载桌面应用 [claude.ai/download](https://claude.ai/download)。
3. 使用个人计划，而非商业计划（我猜是合规要求太多）。
4. 将账户升级到Pro（20美元）或Max（100美元）。这很值得。
5. 进入设置（左下角），开启这些功能：

> 更新：他们现在已为团队提供Claude Computer。

![点击你的名字（左下角），然后点击设置。](https://pbs.twimg.com/media/HEPddG1aAAAEglP.jpg)
![进入桌面应用 > 通用 > 开启浏览器使用 > 开启电脑使用。](https://pbs.twimg.com/media/HEPdhpHaIAAZ3YU.jpg)
![开启电脑使用时，必须点击“开启”来接受。](https://pbs.twimg.com/media/HEPdltUbkAA3Wsa.jpg)
![你可以拒绝某些应用。我拒绝了1Password（里面有我的密码）。](https://pbs.twimg.com/media/HEPdquZXYAAZw_K.jpg)
![我还点击了辅助功能和屏幕录制，授予Claude访问权限。](https://pbs.twimg.com/media/HEPdvEuWcAA8U1P.jpg)

完成后，通过Claude Dispatch将你的电脑连接到手机：

1. 在Claude桌面应用的左侧菜单栏中。
2. 点击Dispatch。然后按照以下步骤操作：

![点击Dispatch。仅限付费个人账户的桌面应用可用。](https://pbs.twimg.com/media/HEPd0DTbwAAPGkC.jpg)
![点击“开始使用”。](https://pbs.twimg.com/media/HEPd4xuXMAAki0s.jpg)
![用手机扫描这个二维码。确保你的手机上也安装了Claude应用，并且登录了同一个账户。它是免费的。](https://pbs.twimg.com/media/HEPd8sFW0AAYJIA.jpg)
![我开启了所有设置来全面测试。然后点击“完成设置”。](https://pbs.twimg.com/media/HEPeA59asAAMkm6.jpg)
![这是我的电脑看到的界面。](https://pbs.twimg.com/media/HEPeE9jaUAA0AHw.jpg)
![这是我的手机看到的界面。完全一样。](https://pbs.twimg.com/media/HEPeK8UbwAE9_0r.jpg)

# 2. 我最喜欢的‘Claude Computer’使用案例

提醒一下，使用Claude Computer（配合手机）需要什么：

1. 你必须安装Claude桌面应用 + 拥有付费个人账户。
2. 你的电脑必须始终保持开机。这就是为什么人们购买Mac mini。
3. 打开手机。打开Claude应用。进入Dispatch。提出要求：

![我的第一个提示：去我的仪表板。给我一些数据。](https://pbs.twimg.com/media/HEPeUNyaoAAGBL7.jpg)

# 3. 如何安排Claude任务

这显然是下一步：为Claude安排任务。

就像……一个向你汇报的员工。

1. 进入左侧边栏，找到“Scheduled”。
2. 创建“New task”。给它一个名称和描述。
3. 最重要的部分：提示词。先测试一下。
4. 选择频率。确保选择正确的模型和文件夹。

![给它一个名称和描述。在粘贴到这里之前，先在普通聊天中测试一下提示词。](https://pbs.twimg.com/media/HEPfRBlXIAA_j6l.jpg)
