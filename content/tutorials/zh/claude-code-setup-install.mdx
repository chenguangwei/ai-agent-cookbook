## 安装此插件后，Claude Code 的体验将截然不同。

https://x.com/Suryanshti777/status/2056022182560665602

Anthropic 悄悄发布了一个名为 `claude-code-setup` 的官方插件，它基本上将 Claude Code 从“还不错”变成了一个真正的 AI 开发环境。

它会扫描你的项目并推荐：

→ 钩子（hooks）
→ 技能（skills）
→ MCP 服务器
→ 子代理（subagents）
→ 自动化（automations）

然后逐步为你设置好一切。

大多数人都在完全原样使用 Claude Code……

这就是为什么他们的体验感觉混乱。

真正的力量来自于其周围的生态系统。

安装：

```bash
/plugin install claude-code-setup@claude-plugins-official
```

在忘记之前收藏此内容。
