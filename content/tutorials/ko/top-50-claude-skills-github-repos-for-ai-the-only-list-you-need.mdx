## 나의 필수 MCP 서버

![](https://pbs.twimg.com/media/HD19jtSboAIYPOg.png)

스킬 외에도 — 저는 이것들을 사용하는 것을 좋아합니다. 스킬은 Claude에게 **방법**을 가르칩니다. MCP는 외부 세계에 대한 **접근**을 제공합니다. 이 세 가지는 제 워크플로우를 완전히 바꿔놓았습니다.

### Tavily

AI 에이전트를 위해 구축된 검색 엔진. 파란 링크가 아닌 깔끔한 구조화된 데이터. 네 가지 도구: 검색, 추출, 크롤링, 매핑. 1분 안에 원격 MCP로 연결.

https://github.com/tavily-ai/tavily-mcp

### Context7

최신 라이브러리 문서를 LLM의 컨텍스트에 주입. 더 이상 환각 API 없음. 더 이상 사용되지 않는 메서드 없음. 프롬프트에 "use context7"을 추가하기만 하면 됩니다. Next.js, React, Supabase, MongoDB, 수천 개 더 지원.

https://github.com/upstash/context7

### Task Master AI

당신의 AI 프로젝트 매니저. PRD 입력 → 의존성이 있는 구조화된 작업 → Claude가 하나씩 실행. 혼란스러운 세션을 적절한 파이프라인으로 전환. Cursor, Claude Code, Windsurf에서 작동.

https://github.com/eyaltoledano/claude-task-master

## 요약

- **스킬** = Claude에게 작업을 더 잘 수행하는 방법을 가르침
- **MCP** = Claude에게 외부 도구 및 데이터에 대한 액세스 권한 부여
