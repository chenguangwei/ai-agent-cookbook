# 200개 이상의 스킬을 2주 동안 시도해보니, 이 8개가 가장 필수적입니다. 마지막 스킬은 꼭 필요합니다

설치하고, 제거하고, 다시 설치했습니다. 2주 동안 이것저것 만져본 결과, 진짜로 필요한 건 이 8개뿐이라는 것을 알게 되었습니다.

기능이 가장 많아서가 아니라, 제 가장 실질적인 고통을 해결해 주기 때문입니다. 더 이상 QR 코드 로그인을 하지 않아도 되고, 반복적인 웹 작업을 하지 않아도 되며, 에이전트가 잊어버릴 걱정도 없고, 여러 에이전트의 상태를 수동으로 관리할 필요도 없습니다.

각 스킬 뒤에 설치 명령어를 적어 두었습니다. 복사해서 사용하세요.

## agent-browser — 에이전트가 웹을 대신 조작하게 하세요

페이지를 스크래핑하고 싶나요? 그냥 말하세요: "[웹사이트]로 가서 [목록]을 가져와." 에이전트가 직접 브라우저를 열고 페이지를 조작한 후 구조화된 데이터를 반환합니다. 환경을 설정하거나, 안티 스크래핑 조치를 처리하거나, 코드를 유지 관리할 필요가 없습니다.

```text
npx skills add vercel-labs/agent-browser@agent-browser
```

## bb-browser — 로그인 없는 마법 도구

예전에는 에이전트가 샤오홍슈에 게시물을 올리려면 먼저 QR 코드를 스캔하여 로그인해야 했고, 세션이 만료되면 다시 스캔해야 했습니다. bb-browser는 로컬 브라우저의 상태를 직접 사용합니다. 이미 로그인되어 있고, 에이전트가 사용자의 신원을 사용하여 작업을 수행합니다. 샤오홍슈에 게시물을 올릴 때마다 QR 코드를 스캔할 필요가 없습니다.

```text
npx skills add epiral/bb-browser@bb-browser
```

## OpenCLI — 18개 플랫폼에서 한 번의 명령으로 작업 수행

댓글 달기, 팔로우, 검색과 같은 수동 작업을 명령줄 인터페이스로 전환합니다. 빌리빌리, 즈후, 샤오홍슈, X, Reddit, GitHub 등 18개 플랫폼을 지원합니다. 또한 스스로 진화하여 새로운 웹 페이지를 CLI로 전환할 수 있습니다.

```text
npx skills add joeseesun/opencli-skill@opencli
```

## skill-vetter — 새 스킬 설치 전 보안 검사

스킬 마켓은 점점 풍부해지고 있지만, 위험도 함께 증가하고 있습니다. skill-vetter는 설치 전에 스캔합니다: 네트워크 외부 호출, 민감한 환경 변수 읽기, 시스템 디렉토리 쓰기, 의심스러운 base64 디코딩 등을 확인합니다.

```text
npx skills add useai-pro/openclaw-skills-security@skill-vetter
```

## self-improving-agent — 에이전트가 반성하며 배우게 하세요

사용자가 수정한 실수, 겪었던 함정, 자주 언급하는 작업 습관을 자동으로 기록합니다. 그런 다음 이를 "지식 카드"로 정리하여 후속 작업에서 능동적으로 사용합니다. 반성하지 않는 에이전트는 아무리 훈련해도 초보자 수준을 벗어나지 못합니다.

```text
npx skills add charon-fan/agent-playbook@self-improving-agent
```

## lossless-claw — 에이전트에게 장기 기억력을 주세요

OpenClaw 작성자가 권장하는 솔루션입니다. 모든 대화를 데이터베이스에 영구 저장한 다음, 백그라운드에서 오래된 대화를 트리 구조의 요약으로 패키징하여 토큰 소비를 제어합니다. 에이전트가 더 이상 "잊어버리지" 않습니다.

```text
npx skills add Martian-Engineering/lossless-claw
```

## control-center — 에이전트 관리 대시보드

여러 에이전트를 관리하는데, 각각 조용해지면 살아 있는지 죽었는지 알 수 없습니다. 이 패널이 문제를 해결합니다: 각 작업이 소비한 토큰 수를 확인하고, 각 에이전트가 정상인지 멈췄는지 확인하며, 각 에이전트의 메모리, 페르소나, 작업 문서를 보고 수정할 수 있습니다.

```text
npx skills add aradotso/trending-skills@openclaw-control-center
```

## openclaw-backup — 에이전트를 위한 최후의 안전망

전체 에이전트의 설정 파일과 메모리 데이터베이스를 정기적으로 백업합니다. 실수로 에이전트를 "죽여도" 복원할 수 있습니다. 설정, 메모리, 과거 대화를 한 번의 클릭으로 롤백합니다.

```text
npx skills add theagentservice/skills@openclaw-backup
```

도구의 가치는 기능이 얼마나 많은지가 아니라, 사용자의 어떤 고통을 해결해 주는지에 있습니다.

이 8개의 스킬은 2주 동안 200개 이상을 시도한 끝에 진짜로 남긴 것들입니다.

도움이 되길 바랍니다.
